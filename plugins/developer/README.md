# developer - Plugin

## Introduction

This ZSH pligin has utility and alias for evironment developer.

## Use

For use open:

```bash    
vi ~/.zshrc
```

And add:

```bash
plugins=(
  ...
  developer
  ...
)
```

## Aliases

|Comand        |Description                                                                              |
|:-------------|:----------------------------------------------------------------------------------------|
|`fzfp`        |Command ***fzf*** and ***bat*** with preview by default.                                 |
