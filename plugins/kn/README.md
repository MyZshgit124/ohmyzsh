# kn - Knative CLI

This plugin provides autocompletion for [kn](https://knative.dev/docs/install/install-kn/) revison, route, service, source, and trigger operations.

To use it, add `kn` to the plugins array of your zshrc file:

```bash
plugins=(... kn)
```
