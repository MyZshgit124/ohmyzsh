# iwhois
### A plugin that provides whois servers using CNAMES with a simple function: `iwhois()`

**Notable websites**

[whois.geek.nz](whois.geek.nz)


**Installation**

To use it, add iwhois to the plugins array of your zshrc file:
```
plugins=(... iwhois)
```


**Functionality reference**

`iwhois()`
