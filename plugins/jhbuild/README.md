# JHBuild

This plugin adds some [JHBuild](https://developer.gnome.org/jhbuild/) aliases.

To use it, add `jhbuild` to the plugins array of your zshrc file:

```zsh
plugins=(... jhbuild)
```

**Maintainer:** [Miguel Vaello](https://github.com/miguxbe)
