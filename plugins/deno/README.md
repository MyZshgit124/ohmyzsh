# deno plugin

This plugin sets up aliases for the common [deno](https://deno.land/) commands

## Aliases

| Alias | Full command |
| --- | --- |
| dr | deno run |
| dh | deno help |
| dbh | deno bundle help |
| dw | deno run --watch |
| dup | deno upgrade |
| di | deno install |
| dt | deno test |
| dl | deno lint |
