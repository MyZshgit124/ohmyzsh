# zsh-current-time-with-zone

Alias plugin for encoding or decoding using `current-time-with-all-zone` or just `ctz` command.

To use it, add `current-time-with-zone` to the plugins array in your zshrc file:

```zsh
plugins=(... zsh-current-time-with-zone)
```
