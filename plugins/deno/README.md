# Deno Plugin

This plugin sets up completion and aliases for [Deno](https://deno.land/).

## Aliases

| Alias | Full command |
| --- | --- |
| db | deno bundle |
| dc | deno compile |
| dca | deno cache |
| dco | deno completions |
| ddoc | deno doc |
| deval | deno eval |
| dfmt | deno fmt |
| dh | deno help |
| dinfo | deno info |
| di | deno install |
| dli | deno lint |
| dre | deno repl |
| dr | deno run |
| drw | deno run --watch |
| dt | deno test |
| dty | deno types |
| dup | deno upgrade |
